# WebProgramming
NJU WebProgramming
共八次作业，6-8以及重构整合的系统在homework6中

一、html+css
提交两个页面，登陆和主页，主页包含至少9张图片和文字信息。主题可以是照片分享/点评，电影介绍等，自拟。自行布局，响应式设计，代码中不可以出现js！！！使用bootstrap仅为及格分数。

二、javascript基础实验
功能：
1. 图片编辑
2. 图片合成
要求：
在作业1基础上进行，功能可参照课堂上描述的完成，也可自行设计,可自行选择结果展示方式。

三、登录与注册
要求：
必须使用nodejs，其他自选。
在前两次作业基础上增加功能。
具体可参照各个知名网站相关内容。

四、 ajax基础实验
通过ajax，一次选择上传单张和多张图片，
需要和之前的编辑与合成功能结合。

五、session and cookie
完成两项功能
1.会话管理，维持登陆状态。
2.安全性。提供机制预防诸如session hijacking等风险。

六、UGC（user generated content, 用户生成的内容）的标签应用 
标签是一种无层次化结构的，用来描述信息的关键词，可以用来描述物品的语义。UGC的标签功能是一种表示用户兴趣和物品语义的重要方式。
目标： 在之前项目中引入标签功能，如给图片进行标注。
应包含如下内容：
1.单一照片/图片可以具有多个标签。
2.可以根据标签检索并显示结果，按照不同属性排序展示（访问量，时间等）
3.用户可自行添加标签
4. 用户可以查看所有标签，及对应的流行度。

七、rest api方式混搭，简单的就可以。

八、重构，历次作业整合到标签系统所做的修改或重构，按照max（重构所做的工作量、标签应用系统）评分。

加分项（包含并不限于）：
1. 标签聚类
2. 智能分类
3. 推荐
